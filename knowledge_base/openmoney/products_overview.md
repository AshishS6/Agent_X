# Open Money Products Overview

## Main Features and Capabilities

Open Money provides a comprehensive business finance operating system that helps companies collect money, pay money, track money, reconcile money, and stay compliant.

---

## 1. Connected Banking

**Description:** Connect your existing current accounts from major banks to Open Money for seamless payment operations and real-time visibility.

### Supported Banks:
- **ICICI Connected Banking**: Direct integration with ICICI Bank current accounts
- **SBI Connected Banking**: Direct integration with State Bank of India current accounts
- **Axis Bank Connected Banking**: Direct integration with Axis Bank current accounts
- **Yes Bank Connected Banking**: Direct integration with Yes Bank current accounts

### Features:
- Secure and direct account-to-account payments
- Real-time balance visibility
- Transaction history sync
- Multi-bank account management
- Digital dashboard for all connected accounts

### Benefits:
- Manage all current accounts in one place
- Get end-to-end overview across banking, payments, and accounting
- Simpler beneficiary management
- Direct payments from connected accounts

## 2. Pay Vendors

**Description:** Streamlined bill management and vendor payment system with direct account-to-account transfers.

### Features:
- **Bill Management**:
  - Upload bills on the go
  - Review and track payables on one dashboard
  - Create bills directly in Open Money
  - Import bills from accounting software

- **Payment Processing**:
  - Make direct account-to-account payments from connected bank accounts
  - Instant payment processing
  - Support for single and bulk payments
  - Payment tracking and status updates

- **Accounting Integration**:
  - Auto-sync bills with accounting software in minutes
  - Two-way sync with Tally, Zoho Books, Oracle NetSuite, Microsoft Dynamics
  - Automatic reconciliation

### Workflow:
1. Add Bill Details - Upload, import, or create vendor bills
2. Pay Bills - Instantly pay bills from connected bank accounts
3. Reconcile Payments - Auto-reconcile payments with bills and sync with accounting software

### Use Cases:
- Vendor payments
- Utility bill payments
- Rent payments
- Service payments
- Supplier payments

## 4. Auto-Reconciliation

**Description:** Automatic reconciliation of payments with bills, invoices, and bank transactions, with seamless accounting software integration.

### Features:
- **Two-Way Sync**:
  - Automatic sync with accounting software
  - Real-time data synchronization
  - Bidirectional data flow

- **Accounting Software Integrations**:
  - **Tally**: Full integration with Tally accounting software
  - **Zoho Books**: Seamless sync with Zoho Books
  - **Oracle NetSuite**: Enterprise-level integration
  - **Microsoft Dynamics**: Microsoft Dynamics integration

- **Reconciliation Capabilities**:
  - Match documents to payments automatically
  - Match payments to bank entries
  - Verify amounts and confirm reconciliation
  - Discrepancy detection and alerts

- **Reconciliation Status**:
  - `pending`: Not yet reconciled
  - `reconciled`: Successfully matched
  - `discrepancy`: Mismatch detected

### Benefits:
- Save time on manual reconciliation
- Reduce errors and discrepancies
- Maintain accurate financial records
- Stay compliant with accounting standards

### Reconciliation Frequency:
- Daily reconciliation for high-volume businesses
- Weekly reconciliation for moderate activity
- Monthly reconciliation for low activity

## 9. Platform Access (Dashboard & Mobile App)

**Description:** Open Money is a fintech operating platform that provides a web dashboard and mobile app for businesses to access all financial management features.

### Important Note:
**Open Money does NOT provide APIs directly.** Open Money is a **platform/dashboard** for accessing features. APIs are provided by **Zwitch**, not Open Money.

### Platform Features:
- **Web Dashboard**: Comprehensive dashboard for accessing all features
- **Mobile App (Android)**: Mobile app for on-the-go financial management
- **Connected Banking**: Connect and manage bank accounts through the platform
- **Invoice Management**: Create and manage invoices through the dashboard
- **Payment Links**: Generate payment links through the app
- **Bill Management**: Upload, manage, and pay bills through the platform
- **Expense Management**: Manage corporate cards and expenses through the app
- **Payroll Management**: Access Open Payroll features through the dashboard
- **Reconciliation**: View and manage reconciliation through the platform
- **GST Compliance**: Access GST features through the app

### Integration:
- **Accounting Software**: Built-in two-way sync with Tally, Zoho Books, NetSuite, Dynamics
- **Banking**: Connected Banking integration with ICICI, SBI, Axis, Yes Bank
- **Payment Gateways**: Integration with payment gateways for collections

### For Developers:
If you need API access for payment processing, banking, or financial services, please refer to **Zwitch's API documentation** - Zwitch provides comprehensive developer APIs.

For detailed information, see [Platform Access](./products/api_solutions.md).
